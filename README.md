# Reactivación JS 2026 - PokéAPI

## 📖 Descripción

Este proyecto forma parte de mi proceso de reactivación como desarrollador Full Stack.

Después de un tiempo alejado de la programación, decidí comenzar nuevamente desde los fundamentos para reforzar conocimientos y volver a construir proyectos con buenas prácticas.

Esta aplicación consume la PokéAPI para buscar información de cualquier Pokémon y mostrar sus principales características de forma dinámica.

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6+)
- Fetch API
- Async / Await
- PokéAPI

---

## ✨ Funcionalidades

- Buscar un Pokémon por nombre.
- Mostrar imágenes frontal y trasera.
- Mostrar altura y peso.
- Mostrar los tipos del Pokémon.
- Manejo de errores cuando el Pokémon no existe.

---

## 🧠 Conceptos practicados

Durante este proyecto reforcé conocimientos sobre:

- Manipulación del DOM.
- Eventos.
- Consumo de APIs REST.
- Promesas.
- Async / Await.
- Métodos de Arrays (`map()` y `join()`).
- Renderizado dinámico con `innerHTML`.
- Manejo de respuestas HTTP.
